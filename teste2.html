Seguem as informações aprendidas hoje:

  • Parágrafo

    • Cria-se um parágrafo com a tag P

  • Lista

    • Cria-se uma lista nao ordenada com a tag UL seguida de LI

    Cria-se uma lista ordenada com a tag OL seguida de LI

  • Imagem

    • Adiciona-se uma imagem com a tag IMG seguida SRC

  • Negrito

  • Utiliza-se a tag b antes da palavra que deverá ser destacada e /b para finalizar.

  • Cmder

    • É um terminal de linha de comandos, como:

  • PWD – mostra pasta atual
  • Mkdir + “texto” – cria uma pasta no diretório atual – (obs.: da mesma forma que criar uma pasta manualmente com o mouse, criar pasta e escrever o nome)
  • Ls – lista o que tem dentro da pasta
  • Cd – entra na pasta
  • Cd.. -volta para a pasta anterior
  • Touch – index.html – cria a página/arquivo index
  • Atom index.html – abre o arquivo index html dentro do atom
